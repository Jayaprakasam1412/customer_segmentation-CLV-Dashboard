# 📊 Customer Segmentation & CLV Dashboard

## 🚀 Project Overview

This project focuses on analyzing customer behavior using **RFM (Recency, Frequency, Monetary)** analysis to identify high-value customers and improve business decision-making.

The solution provides an **end-to-end analytics pipeline** from raw data processing to an interactive dashboard.

---

## 🎯 Business Problem

* Generic marketing campaigns with low effectiveness
* No identification of high-value customers
* Lack of visibility into customer retention risks
* Need for data-driven decision making

---

## 💡 Solution

* Implemented **RFM segmentation model**
* Classified customers into segments like:

  * Champions
  * Loyal Customers
  * At Risk
  * Others
* Built an **interactive Power BI dashboard**
* Enabled business insights for targeted marketing

---

## 🏗️ Architecture

Raw Data → SQL → Python (RFM Model) → CSV → Power BI Dashboard

---

## 🛠️ Tech Stack

* **SQL** → Data cleaning & transformation
* **Python (Pandas)** → RFM calculation & segmentation
* **Power BI** → Data visualization & dashboard

---

## 📊 Dashboard Features

* KPI Cards (Total Customers, Revenue, Avg Value)
* RFM Scatter Plot (Customer Segmentation)
* Revenue Contribution Analysis
* Customer Distribution by Segment
* Interactive Filters (Segment, Recency, Frequency)

---

## 📈 Key Insights

* Champions contribute the highest revenue
* At Risk customers need retention strategies
* Loyal customers provide consistent revenue
* Data-driven segmentation improves marketing efficiency

---

## ⚙️ Automation

* Python script scheduled using Task Scheduler
* Automated weekly data refresh
* End-to-end pipeline execution
